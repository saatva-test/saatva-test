**To run locally:**

1. Install Node.js v14 (https://nodejs.org/en/)
2. ```git clone https://github.com/saatva-test/saatva-test.git```
3. ```cd saatva-test```
4. ```npm install```
5. ```npm start```

**To test:**

```npm test```
